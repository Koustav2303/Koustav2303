### 👑 The Architect Behind the Screen

I am a Frontend Developer based in Bangalore who treats the browser as a limitless canvas. My philosophy is simple: **clean architecture is the baseline; delivering a god-level, immersive visual experience is the standard.** 

From building secure, state-heavy React dashboards to rendering real-time, audio-reactive 3D environments, I engineer interfaces that leave a lasting impression. I have a deep appreciation for high-contrast, cinematic aesthetics, and hyper-realistic editorial design, which heavily influences the visual direction of my code.

<details>
  <summary><kbd>✨ DECRYPT: My Engineering Philosophy</kbd></summary>
  <br>
  <blockquote>
    <ul>
      <li><b>The Aesthetic:</b> I favor modern, high-contrast, cinematic aesthetics. Think Glassmorphism, Neumorphism, and deep, layered, tactile UI.</li>
      <li><b>The Motion:</b> A static site is a dead site. I use tools like GSAP and Three.js to orchestrate fluid micro-interactions, scroll animations, and WebGL layers that make interfaces breathe.</li>
      <li><b>The Architecture:</b> Built to scale. Using React and Tailwind CSS v3, my components are modular, reusable, and heavily optimized for production.</li>
    </ul>
  </blockquote>
</details>
