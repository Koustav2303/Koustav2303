### 👑 The Architect Behind the Screen

I am a Developer based in Bangalore who treats the browser as a limitless canvas. While my foundation is built on heavy, algorithmic logic, my true passion lies in **cinematic frontend architecture**. 

I bridge the gap between secure, robust backend systems and breathtaking, high-fidelity visual design. Whether I'm building state-heavy React dashboards, spinning up Node.js backends, or rendering audio-reactive 3D environments, I engineer digital experiences that leave a lasting impression.

<details>
  <summary><kbd>✨ DECRYPT: My Engineering Philosophy</kbd></summary>
  <br>
  <blockquote>
    <ul>
      <li><b>The Aesthetic:</b> Deep, layered, and tactile. I favor high-contrast, modern UI using Tailwind and custom CSS to craft Glassmorphism and Neumorphism environments.</li>
      <li><b>The Motion:</b> A static DOM is a dead DOM. I orchestrate fluid micro-interactions and scroll animations using GSAP and Three.js to make interfaces breathe.</li>
      <li><b>The Architecture:</b> Built to scale from the server to the screen. Utilizing Node/Express for the backend and modular React components on the frontend.</li>
    </ul>
  </blockquote>
</details>
