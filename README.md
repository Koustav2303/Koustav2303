### 👑 System Architect & Creative Developer

I am a Bangalore-based Software Engineer holding a B-Tech in Computer Science, treating the browser as a high-performance rendering engine. While my foundation is deeply rooted in algorithmic logic and heavy backend systems, my specialization lies in **cinematic frontend architecture** and **full-stack scalability**. 

I bridge the gap between secure, data-intensive backend architectures and breathtaking, high-fidelity visual design. Currently expanding my enterprise capabilities through advanced Java Fullstack architecture (Spring, Hibernate, MySQL) while continuing to push the limits of WebGL and React on the client side.

<details>
  <summary><kbd>✨ DECRYPT: Engineering & Architecture Philosophy</kbd></summary>
  <br>
  <blockquote>
    <ul>
      <li><b>Performance Optimization:</b> Code is written for the GPU. I prioritize 60FPS rendering, utilizing GPU-accelerated CSS transforms and <code>requestAnimationFrame</code> driven loops to ensure zero layout thrashing during heavy animations.</li>
      <li><b>The Aesthetic Engine:</b> A static DOM is a dead DOM. I favor high-contrast, cinematic UI utilizing Tailwind CSS v3 to craft Glassmorphism, Neumorphism, and brutalist design environments that react tactilely to user input.</li>
      <li><b>Scalable Topologies:</b> Built to scale from the database cluster to the screen. I engineer modular, state-managed React components on the frontend, powered by robust Node/Express or Java/Spring backends.</li>
      <li><b>Motion Orchestration:</b> Beyond simple CSS transitions. I use GSAP for complex animation timelines and Three.js for real-time, physics-based 3D rendering.</li>
    </ul>
  </blockquote>
</details>
