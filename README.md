### 👑 The Architect Behind the Screen

I am a Frontend Developer based in Bangalore who treats the browser as a limitless canvas. I specialize in bridging the gap between heavy, complex application logic and breathtaking, high-end visual design. My philosophy is simple: **clean code is expected, but delivering a cinematic, god-level visual experience is the ultimate goal.** 

From building secure, state-heavy dashboards to rendering real-time 3D audio visualizers in the DOM, I build interfaces that leave a lasting impression. 

<details>
  <summary><b>✨ My Design & Development Philosophy (Click to Expand)</b></summary>
  <br>
  <ul>
    <li><b>The Aesthetic:</b> I heavily favor modern, high-contrast, cinematic aesthetics. I frequently utilize <i>Glassmorphism</i> and <i>Neumorphism</i> to create deep, layered, and tactile user interfaces.</li>
    <li><b>The Motion:</b> A static site is a dead site. I leverage tools like GSAP to orchestrate fluid, highly-timed micro-interactions and scroll animations that make interfaces feel alive.</li>
    <li><b>The Architecture:</b> I build with scale in mind. Using React and Tailwind CSS v3, I ensure my components are modular, reusable, and highly performant.</li>
  </ul>
</details>
