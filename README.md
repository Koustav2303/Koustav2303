Frontend Developer
------------------

My name is Koustav Pan. I began my journey in web development by learning the fundamentals of HTML, CSS, and JavaScript, where I developed a strong understanding of how modern websites are structured, styled, and made interactive. During this early stage, I focused on building simple web pages and improving my core front-end development skills.

Currently, I work as a frontend developer, specializing in modern technologies such as React, Tailwind CSS, and Bootstrap. I enjoy creating responsive, user-friendly, and visually appealing web interfaces. By combining React's component-based architecture with the flexibility of Tailwind CSS and the reliability of Bootstrap, I focus on building scalable and efficient web applications.

I am passionate about continuously improving my development skills, exploring new frontend tools and frameworks, and building high-quality web experiences that provide both functionality and great user experience.

* 🌍  I'm based in Bangalore,India.
* ✉️  You can contact me at [pankoustav@gmail.com](mailto:pankoustav@gmail.com)
* 🧠  I'm currently learning Java Fullstack.
